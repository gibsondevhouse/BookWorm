# Beta: Feature Set

1. **Collaboration tools** — Comment threads attached to entities and chapters; ability for editors to propose changes; review queue for authors; accept/reject interface.
2. **Release review & diffing** — Tool to compare two releases or revisions. Highlights changed fields, added/removed relations, and text differences. Allows browsing history.
3. **Search tuning** — Implementation of fuzzy search (typo tolerance), synonyms and alias indexing. Ranking adjustments to prioritise recent edits, canonical entries and user preference signals.
4. **Continuity engine expansion** — Additional rules covering travel plausibility, age consistency, title inheritance, magic system rules and timeline overlaps. A status dashboard summarises open issues by severity and type.
5. **Import/export maturity** — Support for zipped packages with Markdown, JSON and media; conflict resolution strategies; detailed logs and rollback on failure. Admin UI to monitor import/export status.
6. **Operations & observability** — Centralised logging via a chosen stack (e.g. Prometheus + Grafana); health checks for each service; backup scripts and cron jobs; restore scripts with verification; basic metrics dashboards.
7. **UX & accessibility improvements** — Refined navigation, improved readability (line height, measure), accessible colour contrasts, keyboard navigation support and reduced cognitive load in admin interfaces.
8. **Feedback integration** — Implementation of improvements and fixes based on user feedback from the MVP; prioritised bug fixes and quality of life enhancements.
