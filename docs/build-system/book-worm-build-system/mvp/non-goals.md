# MVP: Non‑Goals

- Implementing advanced editor collaboration tools such as inline comments, suggestion mode or real‑time co‑editing; those are reserved for Beta.
- Comprehensive visual map authoring tools; the atlas may use static images and basic markers only.
- Complex relationship graphs or visualisation beyond simple lists of related entities.
- Full AI‑driven assistants or automated tagging; these belong in future roadmaps.
- Deep continuity validations (e.g. travel time calculations, inheritance chains); only baseline checks are included now.
- Performance optimisation or scaling beyond a single instance; Beta and v1 will address performance.
- Multi‑tenant or SaaS capabilities; the MVP targets self‑hosting for a single author.
