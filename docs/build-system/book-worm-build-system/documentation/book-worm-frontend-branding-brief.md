# Frontend & Branding Brief: Book Worm

The branding of Book Worm should evoke a luxury literary archive. The interface must feel like a curated, premium reading product rather than a developer tool or a fandom wiki. Key design principles include:

- **Editorial Aesthetic:** Typography and layout carry the brand. Use a refined serif for headings and a clean sans for interface text. Prioritise text hierarchy, line length and spacing. Avoid flashy UI elements.
- **Cultured & Quietly Premium:** The colour palette should consist of warm neutrals (bone, charcoal, graphite) with one restrained accent (deep forest green, dark burgundy or muted navy). Dark mode should invert neutrals while retaining mood.
- **Tactile & Archival:** Design should evoke printed materials—thick paper, engraved illustrations, map plates. Images should be sparse and curated (maps, sigils, portraits), not generic fantasy art.
- **Mysterious but Controlled:** Provide hints of depth and narrative intrigue without theatrical fantasy tropes. Use subtle textures and elegant motion for page transitions (fades, slides).
- **Navigation Language:** Use human labels (Codex, Timeline, Atlas, Reader, Archive). Avoid technical jargon (Entities, Data, Objects, CMS). Spoiler controls should be labelled as Spoiler‑Free, Mild Spoilers, Full Archive.
- **Public vs Admin:** The public site is spacious and typographic; the admin is denser but must preserve the same palette and typography.
- **Motion:** Animations should be restrained and evoke the feeling of pages turning or drawers opening. Avoid springy or bouncy motion.
- **Microcopy:** Use clear, literary language (Enter the Archive, View Canon Status). Avoid exposing technical terms (Manage Entities, Sync Data).

The brief also defines colour tokens, typography scale, spacing rules, shadows, radii and motion guidelines. It emphasises the need for a single accent colour and a restrained palette. It prohibits cartoonish worm logos; the identity should be a refined wordmark or abstract symbol related to pages, spirals or archives.
