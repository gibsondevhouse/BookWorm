# Dev‑Use: Goals

- Establish the project repository, directory structure and local runtime configuration for development and self‑hosting.
- Select and configure the primary technology stack (e.g. Node/TypeScript backend, React/Next.js frontend, PostgreSQL database).
- Implement basic authentication and role definitions (Public, Editor, Author/Admin) with session management.
- Define the database schema for core entity tables and seed a few example records.
- Scaffold the design system by implementing type scale, spacing scale, colour tokens and base components in a UI kit.
- Build admin CRUD interfaces for a minimal subset of entity types (e.g. Character, Faction, Location, Event) with validation.
- Create placeholder public pages (codex index pages) using the design system to test typography and layouts.
- Deliver initial import/export utilities that can read/write Markdown with front‑matter for these entities.
