# v1: Non‑Goals

- Adding new feature categories not previously planned. The focus is on finishing existing features and optimising them.
- Major design changes or rebranding. Only minor polish and necessary fixes are allowed.
- Expanding to multi‑tenant or SaaS deployments. v1 remains single tenant and self‑hosted.
- Implementing large new continuity rules. Additional rules may be deferred to post‑v1 updates.
- Undertaking a redesign of the collaboration system. Editor workflows will be improved incrementally after launch.
