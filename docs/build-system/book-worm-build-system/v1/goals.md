# v1: Goals

- Optimise performance of the backend, search engine and frontend rendering. Conduct load tests and address bottlenecks.
- Audit the entire permission system and release logic for correctness, ensuring there are no leaks of private or spoiler content.
- Complete all design tasks, including final typography, spacing, colour refinements, motion details and dark mode adjustments. Ensure the public site and admin share a cohesive visual identity.
- Finalise content: update copy, remove placeholder text, complete user onboarding materials and legal pages (licence, privacy, spoiler policy).
- Conduct a comprehensive security audit. Fix vulnerabilities in authentication, data access, third‑party dependencies and code injection risks.
- Prepare the deployment process for production and create installation and upgrade guides. Provide version numbering and a changelog.
- Establish post‑launch guardrails: monitoring, alerting, bug triage process and a plan for patch releases.
