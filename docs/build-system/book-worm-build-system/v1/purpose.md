# v1: Purpose

The **v1** phase delivers the first public release of Book Worm. Its purpose is to finish the product by improving performance, hardening security and permissions, completing the design and content polish, auditing releases and continuity, and implementing guardrails for post‑launch maintenance. v1 focuses on stability and quality rather than new features; it aims to make Book Worm ready for a real author to launch their lore site and for the maintainers to support it confidently.
