# Stage 04: Public Coverage and Verification

## Purpose
Extend the public surface and verification path so the broader Phase 1 content model is still proven through active-release reads rather than only through admin tooling.

## Parts
1. Expanded Public Read Paths
2. Public Discovery Baseline
3. Phase 1 Verification Gate

## Outputs
- public read coverage for the new Phase 1 content types
- minimal discovery or index behavior for release-bound public content
- verification artifacts demonstrating broader release-safe behavior

## Exit Criteria
- the newly supported content types resolve publicly only through the active release
- public discovery surfaces do not leak unreleased draft content
- the Phase 1 verification gate demonstrates broader multi-entity correctness
