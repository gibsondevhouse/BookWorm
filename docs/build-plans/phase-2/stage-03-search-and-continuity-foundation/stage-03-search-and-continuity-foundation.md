# Stage 03: Search and Continuity Foundation

## Purpose

Add the MVP discovery and consistency subsystems while preserving strict release, visibility, and spoiler guarantees.

## Parts

1. Search Index Contract
2. Search API and Public Surface
3. Continuity Issue Baseline

## Outputs

- search indexing rules for the MVP content set
- public and admin-facing search behavior with safe filtering
- baseline continuity checks and issue tracking

## Exit Criteria

- searchable content is derived from allowed release data rather than from drafts
- search filters respect visibility and spoiler rules
- continuity issues can be created, listed, and used during release readiness checks
